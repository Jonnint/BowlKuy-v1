# Cara Pakai WriteDoc.ai untuk Bikin Proposal

## Step 1: Buka WriteDoc.ai
1. Buka browser, ke https://writedoc.ai
2. Login atau daftar kalau belum punya akun

## Step 2: Buat Dokumen Baru
1. Klik "New Document" atau "Create Document"
2. Pilih template "Academic Paper" atau "Proposal" (kalau ada)
3. Kalau gak ada template, pilih "Blank Document"

## Step 3: Copy Paste Prompt
1. Buka file `WRITEDOC_PROMPT.txt` yang udah saya buatin
2. Copy SEMUA isi file tersebut (Ctrl+A, Ctrl+C)
3. Paste ke WriteDoc.ai di bagian prompt/instruction

## Step 4: Generate Dokumen
1. Klik tombol "Generate" atau "Create Document"
2. Tunggu beberapa detik sampai AI selesai generate
3. WriteDoc.ai akan otomatis bikin:
   - Halaman judul yang rapi
   - Daftar isi dengan nomor halaman otomatis
   - Semua BAB dan sub-BAB terstruktur
   - Tabel-tabel yang rapi
   - Nomor halaman (romawi untuk awal, angka untuk isi)

## Step 5: Review & Edit
1. Cek hasilnya, pastikan semua sesuai
2. Kalau ada yang kurang, bisa edit manual atau minta AI generate ulang
3. Bisa tambah gambar/screenshot di bagian Lampiran

## Step 6: Download
1. Klik "Download" atau "Export"
2. Pilih format: Word (.docx) atau PDF
3. Save ke komputer

## Tips:
- Kalau hasil kurang memuaskan, coba generate ulang dengan klik "Regenerate"
- Bisa minta AI untuk adjust formatting dengan command tambahan
- Kalau tabel gak rapi, bisa minta "make tables more structured"
- Untuk nomor halaman, pastikan pilih opsi "automatic page numbering"

## Alternatif Kalau WriteDoc.ai Gak Work:

### Opsi 1: Google Docs + Add-on
1. Buka Google Docs
2. Install add-on "Docs AI" atau "AI Writer"
3. Paste prompt yang sama
4. Generate

### Opsi 2: Microsoft Word + Copilot
1. Buka Microsoft Word (versi 365 yang ada Copilot)
2. Aktifkan Copilot
3. Paste prompt
4. Minta Copilot generate dokumen

### Opsi 3: ChatGPT + Manual Format
1. Paste prompt ke ChatGPT
2. Copy hasil ke Word
3. Format manual (daftar isi, nomor halaman, dll)

### Opsi 4: Notion AI
1. Buka Notion
2. Buat page baru
3. Aktifkan Notion AI
4. Paste prompt
5. Export ke Word

## Kalau Mau Lebih Cepat:
Langsung aja paste isi file `WRITEDOC_PROMPT.txt` ke WriteDoc.ai, nanti dia otomatis bikin semuanya termasuk:
- ✓ Daftar isi otomatis
- ✓ Nomor halaman
- ✓ Tabel rapi
- ✓ Format akademik
- ✓ Struktur BAB lengkap

Gak perlu ribet format manual lagi!
